# KNN Cholesterol Predictor (Regression)

## 📌 Project Overview
This project implements a K-Nearest Neighbors (KNN) Regression model to predict cholesterol levels based on health parameters such as age, blood pressure, heart rate, and exercise habits.

The model is trained using machine learning techniques and includes data preprocessing, model training, evaluation, and a simple web interface using Gradio.

---

## 📊 Dataset Features

The dataset includes the following features:

- Age
- Blood Pressure
- Max Heart Rate
- Exercise Hours Per Week
- Cholesterol (Target Variable)

---

## ⚙️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Gradio

---

## 🤖 Machine Learning Model

Algorithm used:
K-Nearest Neighbors Regressor (KNN)

Steps performed in this project:

1. Data loading and preprocessing
2. Handling missing values
3. Feature selection
4. Train-test split
5. Feature scaling
6. Model training
7. Model evaluation
8. Visualization of results

---

## 📈 Model Performance

Evaluation metrics used:

- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

Example output:

- RMSE: 50.22
- R² Score: 0.06

---

## 🌐 Interactive Prediction Interface

A simple Gradio interface is used to allow users to input:

- Age
- Blood Pressure
- Max Heart Rate
- Exercise Hours Per Week
- K value

The system then predicts the cholesterol level.
